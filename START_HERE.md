## 📞 SUPPORT & TROUBLESHOOTING

### If Something Goes Wrong

```
PROBLEM: Dashboard won't load
SOLUTION: 
  1. Check internet connection
  2. Try different browser
  3. Clear cache (Ctrl+Shift+Del)
  4. Reload page (Ctrl+F5)
  
TIME TO FIX: 1-5 minutes

PROBLEM: Data not showing
SOLUTION:
  1. Verify database file uploaded
  2. Check data exists in raw_data/
  3. Run pipeline.py locally
  4. Reload page
  
TIME TO FIX: 5-10 minutes

PROBLEM: Charts loading slowly
SOLUTION:
  1. Wait 2-3 minutes after deployment
  2. Refresh page (F5)
  3. Check internet speed
  4. Upgrade Streamlit tier if needed
  
TIME TO FIX: 2-5 minutes
```
